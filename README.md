# Unit-4-GUI-Application-Project

Project Introduction/Overview
-Game where you must avoid mobs in order to collect coins in timely manner. Goal is to colelct most amount of coins possible


How to run the program
- Run through LoginPage.java where you must register "Username" & "Password". After registering information login to begin gameplay where you use keybinds W, A ,S, & D or arrow keys to control the movement of your character. A timer countsdown the time you have to collect the coin in order to reset the timer you must collect the coin in a timely matter.

Project goals or purpose
- Goal is to avoid the collsion with the frame of the window and opposing blocks in order to collide with coins (yellow blocks) which increase your score and reset your timer.Try to survive as long as possible and
  collect the highest amount of coins possible
